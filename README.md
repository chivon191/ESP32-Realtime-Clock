# ESP32 Realtime Clock (NTP Server)
## Introduce
Using ESP32 connected to WiFi to fetch time, weather data from the internet, and display them on an OLED screen. This ensures absolute accuracy of the displayed clock time. Additionally, different time zones can be adjusted via the Blynk IoT application on a mobile phone.
## Features
- Use the NTP (Network Time Protocol) server to fetch time from the Internet.
- Display absolutely accurate time, which can be applied to real-time embedded systems.
- Convert time zones using an app on a mobile phone.
## Hardware
- ESP32
- Oled SH1106 1.3 inch
## IDE
- Arduino IDE
- Blynk Control
